Cette page vous permettra de consulter et d'enregistrer les histogrammes des données issues des traitement précédents.
Un premier menu vous permet de choisir l'histogramme à afficher. Il indique aussi par un tiret précédant le nom que l'histogramme sera enregistré sur le disque.
Un champ permet de régler le nombre de classes dans lesquelles seront ventilées les données de type réel. Il est sans effet sur les histogrammes de valeurs entières.
Un menu permet de régler la police de caractère qui sera utilisée dans les schémas.
Deux cases à cocher permettent d'enregistrer l'histogramme affiché sous forme d'image ou de fichier de données brutes CSV.
Les images peuvent être enregistrées en 72dpi pour une utilisation sur écran ou en 300dpi pour un document imprimé.
Lorsque vous en avez terminé avec les histogrammes, cliquez sur la flèche verte à droite du titre.
